# 🚀 Allow In Vera

A **Allow In Vera** é uma startup em fase inicial focada em soluções de tecnologia, consultoria e serviços especializados em infraestrutura, sistemas e monitoramento.

Atualmente, o projeto conta com uma landing page inicial, que representa o primeiro passo da plataforma.

---

## 🧭 Visão do Projeto

Ser uma empresa especializada em:

* Sistemas de Gestão
* Monitoramento de Infraestrutura
* Administração de Servidores e VMs
* Consultoria em TI e DevOps
* Soluções personalizadas para empresas

---

## 💡 Sobre a Landing Page

A versão atual do projeto é uma **landing page de apresentação**, criada com o objetivo de:

* Apresentar os serviços da startup
* Validar a ideia de mercado
* Atrair os primeiros clientes
* Servir como base para evolução do produto

---

## 🛠️ Serviços futuros (Roadmap)

A plataforma evoluirá para incluir:

* Consultoria em infraestrutura e sistemas
* Monitoramento de VMs e servidores
* Painéis de observabilidade (Grafana / Zabbix)
* Automação de ambientes Linux e Docker
* Soluções para gestão de TI empresarial
* Área de clientes e suporte

---

## ⚙️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Design responsivo
* Foco em UI/UX moderno e conversão

## 🎯 Objetivo

Construir uma empresa sólida de tecnologia voltada para performance, automação e gestão de sistemas, ajudando empresas a terem mais controle, eficiência e estabilidade em seus ambientes.
